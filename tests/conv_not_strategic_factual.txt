[Turn 1] USER:
What's the difference between TCP and UDP?

[Turn 1] ASSISTANT:
TCP (Transmission Control Protocol) is connection-oriented — it establishes a connection, guarantees delivery, and maintains packet order. It uses a three-way handshake and includes error checking and retransmission.

UDP (User Datagram Protocol) is connectionless — it sends packets without establishing a connection, doesn't guarantee delivery or order, and has no built-in error recovery. It's faster because of less overhead.

Use TCP when reliability matters (web pages, file transfers, email). Use UDP when speed matters more than reliability (video streaming, gaming, DNS queries).
